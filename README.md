# ScrollLock-Hotkey
When enable ScrollLock then turn on Macro Key

### Before Using: Install <a href="https://www.autohotkey.com/">Autohotkey</a>  



### Default hotkey:
* <b>c: </b> Open Chrome  
* <b>f: </b> Open Facebook  
* <b>d: </b> Open Download  
* <b>t: </b> set Windows on top  
* <b>Numpad + : </b>Volume up  
* <b>Numpad - : </b>Volume down
